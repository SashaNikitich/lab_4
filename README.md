## Опис реалізації

Програма написана на мові **C#** (.NET) і демонструє роботу з масивами об'єктів, сортування за декількома критеріями та юніт-тестування.

### 1. Клас NPC

Об'єктна модель неігрового персонажа містить 5 полів:

* `Name` (string) — ім'я;
* `Surename` (string) — прізвище;
* `Hobby` (string) — хобі;
* `XP` (int) — очки досвіду;
* `Age` (int) — вік.

Реалізовано метод `Equals(NPC guy)` для пооб'єктного порівняння всіх полів та перевизначено `ToString()` для зручного виводу таблицею.

### 2. Сортування та Пошук

У виконавчому класі `SortNPC` реалізовано:

* **Сортування:** за допомогою LINQ (`OrderBy` та `ThenByDescending`).
* Першочергово: **Age** (за зростанням).
* Вторинно: **XP** (за спаданням).


* **Пошук:** Використовується метод `Array.Find` разом із логікою `Equals` для ідентифікації конкретного об'єкта в масиві.

### 3. Тестування (Unit Tests)

Додано проект з тестами на базі **NUnit**, що перевіряють:

* Коректність ініціалізації об'єктів через конструктор.
* Роботу методу порівняння `Equals`.
* Логіку подвійного сортування (перевірка пріоритетності полів).
